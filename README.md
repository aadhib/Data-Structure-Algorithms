## How to run - Ubuntu
1. Clone the repository 
```
git clone https://github.com/aadhib/Data-Structure-Algorithms.git
```
2. Get into the folder
```
cd Data-Structure-Algorithms-main
cd Data-Structure-Algorithms
```
3. Compile the C program 
```
gcc pgmname.c
```
4. Run the C program
```
./a.out
```
